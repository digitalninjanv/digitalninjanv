## Featured Open Source Project

### Aplikasi Bank Sampah Digital

[![Repository](https://img.shields.io/badge/View_Repository-GitHub-black?style=for-the-badge\&logo=github)](https://github.com/digitalninjanv/aplikasi-bank-sampah-php)
![Stars](https://img.shields.io/github/stars/digitalninjanv/aplikasi-bank-sampah-php?style=for-the-badge)
![Forks](https://img.shields.io/github/forks/digitalninjanv/aplikasi-bank-sampah-php?style=for-the-badge)
![License](https://img.shields.io/github/license/digitalninjanv/aplikasi-bank-sampah-php?style=for-the-badge)

A free and open source digital waste bank management system built with PHP Native and MySQL/MariaDB.

This project is designed to help communities, schools, local organizations, and small waste bank operators manage their data more efficiently.

Core features include:

* Citizen/member data management
* Waste deposit transactions
* Balance withdrawal transactions
* Daily and monthly reports
* Excel export
* Public balance and transaction checking page
* Responsive interface
* Modular project structure
* Basic security practices such as prepared statements, output sanitization, and password hashing

The goal is to make digital operational tools more accessible for small organizations that need simple, affordable, and maintainable software.
